📦 Instalasi

Mulai gunakan dalam 2 langkah mudah.

Dengan npm

npm install --save-dev code-formatter

Dengan yarn

yarn add --dev code-formatter

Gunakan CLI

npx code-formatter --write ./src/**/*.js

💡 Atau gunakan VS Code extension untuk format otomatis saat menyimpan file.
